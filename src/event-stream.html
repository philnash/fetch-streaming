Event streaming with fetch

This is an example of different ways of loading data onto a page with fetch.

On the left, the response is gathered using response.text() and then printed to the page. On the right, the body is used as a ReadableStream and printed to the page as it arrives.

In this example, the right hand stream is also delivered as server-sent events using Content-Type: text/event-stream and then parsed in the front end.

Check out the source on GitHub and for more detail on implementing fetch check out the blog post on how using fetch with the streams API gets you faster UX with GenAI apps.
